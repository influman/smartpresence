# Installation
Simulation de présence intelligente automatique
  
     
### Les principes
  
smartPresence apprend de vos habitudes quotidiennes.  
Lors de vos absences, il réplique l'allumage des lumières ou l'ouverture des volets, pour simuler votre présence.  
Il faut attendre une semaine complète en mode "Inactif" avant de l'activer.
  
  
### Ajout des périphériques
Cliquez sur "Configuration" / "Ajouter ou supprimer un périphérique" / "Store eedomus" / "smartPresence" / "Créer"  

  
*Voici les différents champs à renseigner:*

* [Obligatoire] - La résidence concernée
* [Obligatoire] - Le 1er périphérique lumière
* [Optionnel]   - le 2nd périphérique lumière
* [Obligatoire] - Les valeurs numériques pour le On et Off des lumières
* [Optionnel]   - Les deux périphériques de volets roulants
* [Optionnel]   - Les valeurs numériques pour l'ouverture et fermeture des volets
* [Optionnel]   - Le plugin ScorePresence
* [Obligatoire] - Votre code API User
* [Obligatoire] - Votre code API Secret
   
Après installation du plugin, vous pourrez rajouter manuellement autant de périphériques lampes (dans VAR1) ou de périphériques volets (dans VAR2) que souhaités.  
  
  
### Utilisation
  
Choisissez au préalable le mode d'utilisation via le périphérique "smartPresence - Mode":  

* Inactif        - Désactivation de la simulation de présence
* Auto           - Activation automatique si absence remontée par le plugin ScorePresence  
* Actif          - Activation manuelle de la simulation de présence (pour lumières et volets)  
* Actif Lumières - Activation manuelle de la simulation de présence (lumières seulement)  
* Actif Volets   - Activation manuelle de la simulation de présence (volets seulement)  
 
Le périphérique "smartPresence - Statut" fournit l'état de la simulation : Mode | Nombre de lampes allumées | Nombre de volets ouverts
